Sobre el proyecto

¿Qué es VicioTV?

VicioTV es una interfaz de streaming completa contenida en un único archivo .html. No requiere servidor, no requiere instalación, no requiere dependencias. Abrís el archivo y ya tenés una experiencia tipo Netflix funcional.

Consume la API pública de TMDB para obtener catálogo actualizado de películas, series y anime, y utiliza múltiples proveedores de embeddings para la reproducción de contenido.

Características
🎬
Catálogo completo

Películas, series y anime con búsqueda, filtros por género y exploración infinita.

🔀
Multi-provider

6 reproductores configurables. Si uno falla, cambiás al siguiente con un click.

👤
Perfiles

Hasta 6 perfiles con avatares personalizados. Favoritos y progreso separados por perfil.

▶️
Continuar viendo

Guarda el progreso de cada episodio y te lleva exactamente donde lo dejaste.

📱
PWA

Instalable en Android e iOS como app nativa desde el navegador.

🎨
Temas personalizables

Cambiá el color principal y el fondo desde el panel del creador en el navbar.

❤️
Lista de favoritos

Guardá lo que querés ver. Persiste en localStorage, sin cuenta necesaria.

📺
Hero dinámico

Portada rotatoria con los trending del momento, trailer y acceso rápido a detalles.

Uso

Cómo usar

No hay build, no hay npm install, no hay nada que configurar. Solo tres pasos:

terminal
# 1. Clonar el repositorio
git clone https://github.com/boch4maN/viciotv.git

# 2. Entrar a la carpeta
cd viciotv

# 3. Abrir en el navegador
open streamapp.html   # macOS
start streamapp.html  # Windows

O simplemente descargá el archivo streamapp.html y abrilo directamente en tu navegador.

Stack técnico

Tecnologías

Vanilla JS Sin frameworks. Todo con JS puro, DOM nativo y fetch. core
TMDB API Catálogo, metadatos, imágenes y datos de episodios. datos
DiceBear API Generación de avatares para perfiles de usuario. avatares
localStorage Perfiles, favoritos, progreso y preferencias. Sin base de datos. storage
Web App Manifest Instalación PWA en dispositivos móviles y desktop. PWA
CSS Variables Sistema de temas dinámico configurable en runtime. theming
Estructura

Un solo archivo

Todo el proyecto vive en streamapp.html. La organización interna es:

streamapp.html
┌─ <head>
│   ├── Variables CSS y reset
│   └── Estilos completos (~2800 líneas)
│
├─ <body>  — HTML estático
│   ├── Pantalla de perfiles
│   ├── Modal editor de perfil
│   ├── Navbar + búsqueda
│   ├── Hero rotatorio
│   ├── Grillas de contenido
│   └── Modal de reproducción
│
└─ <script>  — Toda la lógica
    ├── Configuración de providers
    ├── Sistema de perfiles
    ├── Integración TMDB
    ├── Reproductor + episodios
    ├── Favoritos y progreso
    └── Temas y PWA
